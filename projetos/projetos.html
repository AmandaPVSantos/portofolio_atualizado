Cadastro e Listagem de Produtos

Este projeto foi desenvolvido com o objetivo de criar uma aplicação web prática e eficiente para o cadastro e a exibição de produtos. O sistema permite a inserção de novos produtos com informações como nome, valor e disponibilidade, e organiza a listagem de maneira automática, ordenada por valor.

A interface intuitiva e responsiva foi construída utilizando HTML, CSS e Bootstrap, garantindo uma experiência de usuário fluida em diferentes dispositivos. Além disso, o JavaScript foi utilizado para manipular dinamicamente a exibição dos dados, oferecendo uma interação ágil e moderna.

Foi utilizado as seguintes tecnologias: HTML, CSS, Bootstrap e JavaScript.

Verificador de Idade

Este projeto foi desenvolvido com o objetivo de calcular a idade de forma simples e direta. Ao inserir o ano de nascimento em um campo numérico, o sistema processa e exibe a idade com base no ano fornecido.

Embora seja um projeto básico, ele é uma ótima ferramenta para reforçar conceitos essenciais de manipulação de dados e interação com o usuário, além de manter a prática ativa.

A simplicidade do projeto o torna uma excelente base para futuras expansões e aprimoramentos.
